# stock_picker
-----------
-----------
This method provides an array of two numbers, representing the best two day to buy and sell for maximum profit, for given array input of stock values. 

Parameters
---------------------------
(array)

Array of numbers representing stock values for a week, month, or any number of given elements greater than two. 

Returns
---------------------------
Array [x,y] where x is to buy and y is to sell. 

Fundementals Employed:
---------------------------
Methods for arrays and enumerables. Use of combination(x) array method and combining this with max_by to get the lowest and highest days to buy and sell respectively. Then finding the index of that in the input array which represents the day. 

What I learnt:
---------------------------
New enumerable methods and making code efficient as possible.
